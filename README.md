# Python Core Learning

A study repository for core Python concepts.

## Overview
This repo collects short notes and runnable notebooks for learning Python as a language.  
It focuses on small concept-based examples rather than large projects.

## Topics
- Variables and values
- Data types
- Input and output
- Conditionals
- Loops
- Functions
- Collections
- Errors and exceptions
- Modules

## Structure
- `01-09`: each topic folder contains `notes.md` and `examples.ipynb`

## Format
- `notes.md`: short concept summaries and review points
- `examples.ipynb`: runnable notebook examples with saved output
